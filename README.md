# GritForge Consultants - Steel Bid Review Service

## Service Overview

Expert structural steel bid review for erectors and fabricators. Fixed-price reviews with 24-48 hour turnaround to catch scope gaps, tonnage errors, and plan conflicts before bid submission.

## Repository Structure

```
gritforge-bid-review/
├── index.html                          # Landing page (deploy to web host)
├── email-templates.txt                 # Outreach templates for clients
├── delivery-report-templates.md        # Report templates for all 3 tiers
├── README.md                           # This file
├── client-projects/                    # Store client project files here
│   └── [project-name]/                 # One folder per client project
│       ├── plans/                      # Client-provided plan PDFs
│       ├── takeoff/                    # Client takeoff data
│       ├── review-notes/               # Your working notes
│       └── deliverables/               # Final reports to send
├── outreach/                           # Marketing and client acquisition
│   └── prospect-tracking.csv           # Track emails sent and responses
└── assets/                             # Images, logos, etc.
```

## Getting Started

### 1. Deploy Landing Page
- Upload `index.html` to web host
- Update email templates with actual URL
- Test all links and contact form

### 2. Initial Outreach
- Start with warm contacts (use Template 1B for Founding Client offer)
- Send to anyone you've done bid work for previously
- Use `outreach/prospect-tracking.csv` to track responses

### 3. Cold Outreach
- Research Texas/Colorado structural steel contractors
- Use Template 2 (Cold Outreach) from email-templates.txt
- Max 10/day to avoid spam filters
- Track open rates and responses

### 4. Delivering Reviews
- Use templates from `delivery-report-templates.md`
- Replace ALL [bracketed placeholders] with actual data
- ALWAYS include the "Kill Switch" section (3 critical issues)
- Deliver as PDF via email

### 5. Follow-up
- Request testimonials after successful bids (Template 5)
- Offer retainer pricing after 2-3 reviews (Template 6)
- Re-engage inactive clients every 60 days (Template 7)

## Client Workflow

### Intake
1. Client emails arron@gritforge.com with project details
2. Confirm tier, pricing, and turnaround timeline
3. Request plan PDFs (or plan room link)
4. Request client's existing takeoff (Tier 2/3)
5. Confirm bid deadline

### Review Process
1. Download/organize plans in `client-projects/[project-name]/plans/`
2. Perform review using tier-appropriate template
3. Document findings in working notes
4. Generate final PDF report
5. Deliver via email before deadline

### Post-Delivery
1. Schedule review call (Tier 2/3)
2. Answer any follow-up questions
3. Request testimonial if bid is successful
4. Track for repeat business

## Important Policies

### Confidentiality
- All plan sets, pricing, and project details are confidential
- Never share client information with anyone
- Delete plans 30 days after project completion

### Liability
- This is a review service, NOT engineering or cost estimation
- Include disclaimer in every report
- Client is responsible for all final decisions

### Service Boundaries
**This service IS:**
- Expert validation and second-eye review
- Risk identification and scope analysis
- Professional bid package preparation (Tier 3)

**This service IS NOT:**
- Liability transfer or bid coverage
- Engineering design or calculations
- Contractual or legal advice
- Decision-making for the client
